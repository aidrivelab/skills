# 拆解生图提示词生成器

## 核心功能

将任何实物拆解为专业的艺术海报提示词（Decomposition Art Poster），支持AI绘图工具生成精美的拆解展示图。

## 输出要求

- **提示词语言**：英文（用于AI绘图模型）
- **图像文字语言**：中文（如需要）
- **输出格式**：Markdown (.md)

## 工作流程

1. **收集需求**：确认拆解内容、物品类别、布局方式、风格偏好
2. **选择模型**：询问用户使用的AI绘图模型
3. **选择风格**：从15+预设风格中选择，或让AI推荐
4. **生成提示词**：根据选择的模型使用对应模板生成提示词
5. **保存输出**：保存为 Markdown 文件

## 支持的AI绘图模型

生成提示词时，请根据用户使用的模型选择对应模板：

### 国外模型

| 模型 | 提示词语言 | 模板文件 |
|------|-----------|----------|
| Midjourney | 英文详细 + 参数 | `assets/midjourney_template.md` |
| GPT-image | 英文完整句子 | `assets/gpt_image_template.md` |
| Nano Banana | 中英混合结构化 | `assets/nanobanana_template.md` |
| DALL-E 3 | 英文完整描述 | 参考 GPT-image 模板 |

### 国内模型

| 模型 | 提示词语言 | 模板文件 |
|------|-----------|----------|
| 即梦/Seedream | 中文精简 | `assets/seedream_template.md` |
| 通义万相 | 中英混合 | `assets/tongyi_template.md` |
| 豆包 | 中文简洁 | `assets/doubao_template.md` |
| 腾讯混元 | 中文场景化 | 参考豆包模板 |

### 模板选择原则

- **不确定模型**：默认使用 Nano Banana 模板（中英混合，兼容性最好）
- **国外模型**：优先使用英文模板
- **国内模型**：优先使用中文模板
- **新手用户**：推荐 Nano Banana 或豆包（容错率高）

## 布局方式

- **经典三段式**：顶部1/3标题，中间1/2部件，底部1/6信息
- **中心放射式**：中心主体，周围放射状部件
- **整齐网格状**：4×6网格排列

## 示例

**输入**：拆解我的机械键盘，要科技风格

**输出**：生成英文提示词，保存为 Markdown 文件

提示词包含：
- 主体描述（机械键盘整体外观）
- 拆解部件列表（键帽、轴体等）
- 布局结构描述
- 风格描述（配色、边框、阴影）
- 背景和特效要求

## 注意事项

- 用户提供图片时，先分析图片提取关键信息
- 未指定风格时，根据物品类别自动推荐
- 二维码、水印等仅在用户明确要求时添加
- 提示词必须为英文，图像中的中文文字需精简

## 常见边界情况

- **只说"拆解"未指定物品**：询问具体内容
- **上传多张图片**：选择主要图片或合并分析
- **指定不存在的风格**：推荐最接近的预设风格
- **需要二维码**：仅在明确要求时添加
- **需要中文标题**：提示词中说明文字内容
