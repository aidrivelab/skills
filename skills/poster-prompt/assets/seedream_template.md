# 即梦/Seedream 提示词模板 / Seedream Prompt Template

## 模型特点

- 字节跳动旗下AI绘画模型
- 中文理解能力强，专为中文设计优化
- 支持中文字体生成
- 提示词遵循能力强

## 提示词结构

```
[中文主体描述] + [风格描述] + [布局说明] + [光效/材质]
```

## 模板示例：拆解海报

### 机械键盘拆解

```
机械键盘拆解海报，展示键帽、轴体、PCB板、外壳、卫星轴等部件
科技工业风格，深蓝色调，科技感
网格整齐排列，工作室灯光
蓝图网格线条，金属质感，霓虹光效
```

### 时尚穿搭拆解

```
时尚穿搭拆解海报，展示上衣、裤子、鞋子、包包、配饰
时尚杂志风格，简约白色背景，金色装饰
优雅排列布局，商业摄影

```

## 写作技巧

1. **简短精炼**：每段不超过20字
2. **关键词明确**：使用模型熟悉的关键词
3. **风格一致**：保持整体风格统一
4. **中英混合**：可适当添加英文质量词

## 输出格式

中文为主，可添加英文质量标签。
