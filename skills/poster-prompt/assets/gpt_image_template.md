# GPT-image 提示词模板 / GPT-image Prompt Template

## 模型特点

- 强大的自然语言理解能力
- 支持复杂文本指令解析
- 融合世界知识库
- 提示词遵循能力强

## 提示词结构

```
A [style] image of [subject] with [components] arranged in [layout]. The design features [color palette] background with [lighting]. Include [additional elements]. The overall mood is [mood].
```

## 模板示例：拆解海报

### 机械键盘拆解

```
A professional tech industrial style decomposition poster of a mechanical keyboard, showing an exploded view with keycaps, switches, PCB board, case, and stabilizers displayed in a clean organized grid. The design has a dark blue gradient background with cyan neon accents and blueprint-style grid lines. Professional studio lighting. The overall mood is modern, technical, and sleek.
```

### 时尚穿搭拆解

```
A high-end fashion magazine style decomposition poster of an outfit, showing an exploded view with tops, pants, shoes, bag and accessories arranged in an elegant horizontal layout. The design features a clean white background with subtle golden accents. Professional commercial photography lighting. The overall mood is luxurious, elegant, and sophisticated.
```

## 写作技巧

### 使用完整句子

```
✅ A decomposition poster showing a mechanical keyboard with its components...
❌ mechanical keyboard, keycaps, switches, PCB
```

### 明确风格描述

```
✅ The design has a modern tech industrial aesthetic with blueprint elements
❌ tech style
```

### 添加氛围描述

```
✅ The overall mood is professional, sleek, and innovative
❌ good looking
```

## 拆解海报关键描述

| 元素 | 英文描述 |
|------|----------|
| 拆解布局 | exploded view, decomposition layout, separated components |
| 网格排列 | organized grid arrangement, clean layout |
| 科技风格 | tech industrial aesthetic, blueprint style |
| 时尚风格 | high-end fashion magazine style, elegant |
| 专业灯光 | professional studio lighting, soft lighting |
| 现代感 | modern, sleek, innovative |

## 输出格式

直接输出英文描述性提示词，使用完整句子。
