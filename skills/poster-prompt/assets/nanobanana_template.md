# Nano Banana 提示词模板 / Nano Banana Prompt Template

## 模型特点

- 谷歌 Gemini 系列图像模型
- 中文支持好，理解中英混合提示词
- 结构化标签效果好
- 支持自然语言描述

## 提示词结构

```
[中文描述主体] + [英文风格标签] + [中文布局说明] + [英文质量标签]
```

## 模板示例：拆解海报

### 机械键盘拆解

```
机械键盘拆解海报，展示键帽、轴体、PCB板、外壳、卫星轴等部件，按网格整齐排列
科技工业风格，深蓝渐变背景，青色霓虹光效
蓝图网格线条，测量尺刻度，十字准星 technical markers
工作室灯光，简洁专业
8k, detailed, professional photography, decomposition poster
```

### 时尚穿搭拆解

```
时尚穿搭拆解海报，展示上衣、裤子、鞋子、包包、配饰，按优雅布局排列
```

## 混合语言技巧

| 元素 | 中文 | 英文标签 |
|------|------|----------|
| 风格 | 科技工业风格 | tech industrial, blueprint |
| 布局 | 网格整齐排列 | grid layout, organized |
| 光效 | 霓虹光效 | neon accents, glow |
| 背景 | 深蓝渐变 | dark gradient, blue |
| 质量 | 8K超清 | 8k, detailed, professional |

## 常用质量标签

```
8k, detailed, professional, high quality, studio lighting
clean background, modern, minimalist
decomposition poster, exploded view, schematic
```
