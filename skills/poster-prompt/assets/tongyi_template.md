# 通义万相提示词模板 / Tongyi Wanxiang Prompt Template

## 模型特点

- 阿里云AI绘画模型
- 中文理解能力强
- 支持中英混合提示词
- 风格多样化

## 提示词结构

```
[中文主体描述] + [风格关键词] + [布局说明] + [英文质量词]
```

## 模板示例：拆解海报

### 机械键盘拆解

```
机械键盘拆解海报，展示键帽、轴体、PCB板、外壳、卫星轴
科技工业风格，深蓝渐变背景，霓虹光效
网格整齐排列，专业的科技感
detailed, 8k, studio lighting, tech industrial
```

### 时尚穿搭拆解

```
时尚穿搭拆解海报，展示上衣、裤子、鞋子、包包、配饰
时尚杂志风格，白色背景，金色点缀
优雅水平布局，高级感
detailed, fashion magazine, elegant, professional
```

## 常用关键词

### 风格

| 中文 | 英文 |
|------|------|
| 科技感 | tech, futuristic |
| 时尚 | fashion, stylish |
| 极简 | minimalist, clean |
| 复古 | vintage, retro |
| 艺术 | artistic, creative |

### 质量

| 英文标签 | 说明 |
|----------|------|
| detailed | 细节丰富 |
| 8k | 高清 |
| professional | 专业 |
| studio lighting | 工作室灯光 |

## 输出格式

中文描述为主，英文质量标签放末尾。
