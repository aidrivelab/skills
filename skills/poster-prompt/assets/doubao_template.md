# 豆包提示词模板 / Doubao Prompt Template

## 模型特点

- 字节跳动旗下AI助手
- 使用Seedream图像生成模型
- 中文理解能力强
- 简洁有效的提示词效果最好

## 提示词结构

```
[中文主体] + [风格] + [布局] + [光效]
```

## 模板示例：拆解海报

### 机械键盘拆解

```
机械键盘拆解海报，键帽、轴体、PCB板、外壳、卫星轴网格排列
科技风格，深蓝背景，霓虹光效
工作室灯光，金属质感
```

### 时尚穿搭拆解

```
时尚穿搭拆解海报，上衣、裤子、鞋子、包包、配饰优雅排列
杂志风格，白色背景，金色点缀
商业摄影，高级感
```

## 常用关键词

### 风格

- 科技感、时尚、简约、复古、可爱

### 布局

- 网格排列、整齐、爆炸图、分解

### 质量

- 8k、细节丰富、专业、干净

## 输出格式

中文简洁描述，每段简短。
