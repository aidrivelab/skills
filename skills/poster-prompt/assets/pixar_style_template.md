## 角色渲染 / CHARACTER RENDERING

### 面部 / Face
- **眼睛 Eyes**: 皮克斯大眼睛，多重高光，虹膜细节，IOR 1.376角膜 / Pixar big eyes, multiple highlights, iris details, IOR 1.376 cornea
- **皮肤 Skin**: 三层SSS次表面散射(0.5mm/1.5mm/3.0mm)，双层高光，菲涅尔效应，可见毛孔 / Three-layer SSS (0.5mm/1.5mm/3.0mm), double specular, Fresnel effect, visible pores

- **特征 Features**: 主副双高光，成簇感，碎发，颜色渐变 / Primary & secondary highlights, clustered feel, flyaways, color gradient

### 服装 / Clothing
- **纹理 Texture**: 微观级织物纹理 / Micro-level fabric texture
- **材质 Material**: 基于物理的PBR材质 / Physics-based PBR material
- **细节 Details**: 物理模拟褶皱，张力点，缝合线细节 / Physics-simulated wrinkles, tension points, stitching details

- **曲线 Curve**: 温和S型曲线增强对比度 / Gentle S-curve to enhance contrast
- **色温 Temperature**: 暖色+200K / 冷色-200K / Warm +200K / Cool -200K
- **饱和度 Saturation**: 整体-5%，重点色彩+10% / Overall -5%, key colors +10%
