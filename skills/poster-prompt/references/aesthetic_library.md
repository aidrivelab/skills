## 🎨 风格套餐（一键选择）

### 套餐1: 时尚精美风

- **阴影**: 精致渐变阴影
- **特效**: 金箔效果 + 暗角

### 套餐4: 可爱皮克斯风

- **配色**: 柔和马卡龙色系

## 🖼️ 版式库（用自然语言描述）

### 版式1: 经典三段式
**描述**: 上面三分之一放标题，中间二分之一展示拆解内容，下面六分之一放签名和品牌

```
┌─────────────────────────┐
│     标题区域 (33%)      │
├─────────────────────────┤
│                         │
│   拆解展示区域 (50%)    │
│                         │
├─────────────────────────┤
│   签名/品牌 (17%)      │
└─────────────────────────┘
```

### 版式2: 上部分栏式
**描述**: 上面三分之一分成三栏，中间二分之一展示物品，下面六分之一放信息

```
┌──────┬──────┬──────┐
│ 栏1  │ 栏2  │ 栏3  │  (33%)
├─────────────────────┤
│                     │
│   拆解展示区域      │  (50%)
│                     │
├─────────────────────┤
│   信息区域          │  (17%)
└─────────────────────┘
```

### 版式3: 中心放射式
**描述**: 中间放主体，周围放射状排列部件

```
    ╱╲
   ╱  ╲
  ╱ 主 ╲
 ╱  体  ╲
╱────────╲
部件围绕中心放射排列
```

### 版式4: 整齐网格状
**描述**: 所有部件整齐排列在网格里，每个格子放一个

```
┌───┬───┬───┬───┐
│ 01│ 02│ 03│ 04│
├───┼───┼───┼───┤
│ 05│ 06│ 07│ 08│

│ 09│ 10│ 11│ 12│
└───┴───┴───┴───┘
```

## 📦 元素包（自动配齐）

### 标题元素包
- 主标题字体: 中文方正宋刻本秀楷 / 英文Playfair Display
- 副标题字体: 飘逸手写体
- 标题效果: 金箔效果 + 新艺术风格装饰框

### 标注元素包
- 部件编号: 圆形数字徽章 (01-36)
- 连接线: 优雅虚线/实线
- 材质标签: "100%真丝"、"意大利小牛皮"样式
- 测量标尺: cm/inch 标记

### 装饰元素包
- 几何框: Art Deco 六边形/圆形细线框
- 十字准星: 四角和关键焦点处
- 材质样本: 底部微距特写方块
- 信息卡片: 优雅边框卡片

### 品牌元素包
- 水印: 半透明手写体 (10-15% 透明度)
- 二维码: 右下角 + Art Deco 边框
- 物品计数: 右上角八角形框
- 时间轴: 底部横跨
