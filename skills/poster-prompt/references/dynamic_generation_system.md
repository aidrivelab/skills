## 🖼️ 第三步：格局理解系统

### 用户可以用自然语言描述格局

**预设格局：**
1. **经典三段式** - 上1/3标题，中1/2展示，下1/6签名
2. **上部分栏式** - 上1/3分三栏，中1/2展示，下1/6信息
3. **中心放射式** - 中间主体，周围放射排列
4. **整齐网格状** - 所有部件放网格里

**用户也可以自定义：**
- "上面一半放标题，下面一半分左右两栏"
- "中间放主体，四周一圈放部件"
- "左上角放标题，右下角放签名，中间全部展示"
- "从上到下依次排列，每件物品占一行"

### AI智能理解格局

不管用户怎么描述，AI都能理解并应用到布局中！
