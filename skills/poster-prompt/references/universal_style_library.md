## 🎨 自定义风格创建系统

### 你可以这样描述风格：

**例子1：**
```
风格：像苹果官网那样的极简风格
- 配色：白色、浅灰、黑色
- 边框：极细或者不要边框
- 阴影：很淡的阴影
- 感觉：干净、现代、高级
```

**例子2：**
```
风格：像80年代复古海报
- 配色：霓虹粉、霓虹蓝、黑色
- 边框：发光的霓虹边框
- 特效：故障艺术效果、扫描线
- 感觉：未来、科技、炫酷
```

**例子3：**
```
风格：像梵高的画
- 配色：丰富的色彩、强烈的对比
- 特效：油画笔触效果
- 感觉：艺术、有创意、有表现力
```

## 🔧 动态生成规范模板

### 基于用户描述的智能生成

当用户描述一个物品时，AI会：
1. 识别物品类别
2. 推荐最适合的2-3个风格
3. 自动配好配色、边框、阴影、特效
4. 生成完整提示词

### 你也可以混合使用风格

```
例子：
拆解：机械键盘
风格：科技工业风，但要加一点赛博朋克的霓虹效果
格局：经典三段式
```

AI会自动混合两种风格，生成你想要的效果！
